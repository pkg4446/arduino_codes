요청사항

슬립모드
5v 인입시(AC인입orUSB인입) io3 인식 슬립모드 중지

가온
목표온도 = 공간온도

ex) 목표온도 25도 설정 시 공간온도 15도 일때 가온 on
1. 가온이 켜지고나서 히터온도는 목표온도로 맞춰 히팅진행한다.
2. 일정 시간 이 후 공간온도의 변화가 없거나 내려가면 히터온도를 상향조정한다(ex 10분 2도)

절대값 : 히터판은 50도를 넘지않는다 / 봉구온도가 35도이상이면 가온을 하지않는다
